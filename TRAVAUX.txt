   - Chargez les images à assembler.
      - Trouvez des points caractéristiques dans chaque image.
      - Faites correspondre les points caractéristiques entre les images.
      - Estimer la matrice d'homographie entre chaque paire d'images.
      - Déformer les images en utilisant les matrices d'homographie pour les aligner.
      - Mélangez les images déformées pour créer un panorama.
      - Enregistrez le panorama dans un fichier.
